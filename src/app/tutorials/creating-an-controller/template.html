angularTrainer object is reference to ng-amd public methods which is published in bootstrap.js 

         /**
         * Publish api to window
         * @type {*}
         */
        window.angularTrainer = bootstrapHelper.publishPublicApi(
            (function () {
                var html = document.getElementsByTagName('html')[0];
                return html.getAttribute('data-require-js-min') === "true";
            }())
        );

package.json

Package json requires a "name" and "type" object.
Name must be same as component name in this case -> creatingAnController
Type must be same as component type in this case -> controller
Menu object is an custom object which im using in at-menu service to create dynamic menu. 

{
    "name": "creatingAnController",
    "type": "controller",
    "since": "0.1.0",
    "version": "0.1.0",
